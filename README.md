# 🧩 LeetCode Solutions

Personal collection of LeetCode problems I solve to keep my problem-solving sharp and track my progress over time.

## Why this repo?

Basically I wanted a place to keep everything organized and look back at how I'm improving. No tutorials, no solution videos — I sit down, think it through, and write the code. Sometimes it takes way longer than it should, but that's the point.

Main language is Python, though I'm planning to add solutions in other languages as I go.

## Structure

```
.
├── easy/
├── medium/
└── hard/
```

Each folder contains solutions grouped by difficulty. File names follow the pattern `<problem_name>.py` (e.g. `Personal_atoi.py`).

## Ground rules I set for myself

- No looking at other people's solutions before solving it myself
- No tutorial videos
- If I'm stuck I can read the problem again, think harder, and maybe sketch something out — but the final solution has to be mine
- Comments in the code explaining my thought process (at least for the less obvious ones)
